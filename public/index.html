.brand — The Seventh Typology

What is this?

A .brand file is not a document. It's an agent.

This page is a .brand agent — a living brand identity powered by semantic AI. The SBB brand isn't stored as a PDF or a style guide. It's an MCP (Model Context Protocol) server that any AI tool can connect to.

When you asked a question above, you queried the brand's semantic memory — 768-dimensional vector embeddings of every guideline, rule, and cultural value. The brand understands what you're asking, not just keyword-matching.

This is the seventh typology of dynamic brand identity — after Conteneur, Transformatif, Génératif, Réactif, Participatif, and Évolutif. The conscious identity. A brand that doesn't just change — it thinks.
